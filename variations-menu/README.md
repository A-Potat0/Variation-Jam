## Description

this project focuses on the idea of crushing. you have different representations of the concept. you have the walls, the cannon and the bouncy ball. the main gameplay is pressing keys on your keyboard to overcome the challenge. whilst one of them has an option for victory most of these challenges are only for the fun of seeing how long you can survive by pressing your keyboard.
